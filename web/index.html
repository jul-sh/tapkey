keytap

Passkeys that turn into real keys.

Register a passkey once, then derive the same SSH key, age identity, or app secret on any device that can unlock it. Everything runs locally — no server ever sees your keys.

How it works

  1. Register a passkey with keytap
  2. Pick a key name and output format (SSH, age, hex, ...)
  3. Unlock your passkey — keytap derives the same key every time

CLI (macOS & Linux)

A native command-line tool that uses Touch ID or platform authenticators for passkey authentication.
